# HappySpaceBirthday


🌌 Интерактивный веб-сайт с плавной анимацией скролла, посвященный галактике NGC 3982.

## Описание
Поздравительный лендинг, который встречает пользователя полноразмерным изображением галактики. При скролле изображение плавно смещается и уменьшается, уступая место текстовому блоку с описанием. Фон дополнен анимированным звездным полем на CSS.

## Технологии
- **Core:** HTML, CSS, Vanilla JavaScript
- **Animations:** GSAP + ScrollTrigger
- **Smooth Scroll:** Lenis

## Запуск
1. Откройте https://minchwithroots.github.io/HappySpaceBirthday/
2. Наслаждайтесь :D
